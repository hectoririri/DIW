Tabla de factores SEO
Factor SEO Checkbox Descripción
1. Título de la página, etiqueta title
Etiqueta que utilizan los buscadores para el título del resultado de búsqueda
2. Descripción de la página, metadescripción o metaetiqueta description
Metaetiqueta description que utilizan los buscadores para el fragmento del resultado de la búsqueda
3. Etiquetas de encabezado
Etiquetas de encabezamiento que se indican por numeración de más a menos importante (h1 a h6)
4. Maquetación sin tablas
Maquetación de la página html con etiquetas html correspondientes y hojas de estilo en cascada CSS
5. Minificar el código HTML
Evitar espacios en blanco, comentarios innecesarios y utilizar herramientas específicas para reducir la velocidad de carga de la página web
6. Número de enlaces internos/externos
Mantener equilibrio entre número de enlaces internos y extenernos para no perder popularidad
7. Anchor text en los enlaces entrantes
Texto de anclaje o anchor text en el que el usuario hace click para entrar a una web. Deben contener palabras clave ya que estas intervienen en el cálculo que realizan los buscadores para el posicionamiento
8. Busca errores en tu documento HTML
Utilizar validador HTML para encontrar errores y solucionarlos. No utilizar etiquetas HTML antiguas y deprecadas
9. Incluye un favicon
Disponer de favicon sirve para mejorar el posicionamiento online de una pagina web. Ayuda a mejorar el estilo de esta
10. Incluye un mapa del sitio
Mapas de sitio mejoran el posicionamiento en los buscadores de un sitio web, ya que asegura que todas las páginas puedan ser encontradas. Así los usuarios acceden a todo el contenido web
11. Utiliza el atributo alt en todas tus imágenes
Etiqueta alt que sirve para saber qué es lo que se visualiza en una imagen, además de mostrar este texto si la imagen no carga
12. Ten en cuenta la accesibilidad web
Mejorar la experiencia del usuario con funciones de accesibilidad. Imágenes accesibles, tablas accesibles, formularios accesibles, etc...
13. Busca mejoras mediante una herramienta especializada
Utilizar herramientas como Semrush, Ahrefs, sistrix, Majestic, etc... Estas mejoran velocida de carga, seguridad, certificado SSl, cantidad de enlaces internos y externos, enlaces rotos, etc...
